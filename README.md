# Лаба 1: Простой терминал на Python с поддержкой основных команд файловой системы.

## Запуск программы
```powershell
 git clone https://github.com/DariEFAL/lab_3_algorithm.git
 cd lab_3_algorithm
 uv venv
 .venv\Scripts\activate
(Для Linux: source .venv/bin/activate)
 python -m src.main функция
```
- Чтобы для сортировок ввести отрицательные числа надо написать -- после команды:
python src/main.py sortbubble -- 43 -2 -5 0 5
